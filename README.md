# Loongson-FPGA Harvest Robot

基于龙芯2K0301 + Cyclone IV FPGA异构计算的智能巡检采摘一体化机器人

## 项目简介

针对农业劳动力短缺、采收成本高、果实损伤率大等问题，设计并实现了一款全自主智能巡检采摘机器人。采用龙芯2K0301作为决策主控（Linux + ROS），Cyclone IV FPGA作为实时执行核心，通过SPI总线协同工作，形成“CPU决策 + FPGA执行”的异构架构。

## 主要性能指标

- 果实识别精度 (mAP): 95.38%
- 单果采摘耗时: 4.67秒
- 果实损伤率: < 0.5%
- 逆运动学求解延迟: 0.2毫秒 (FPGA硬件加速)
- 可替代人工比例: > 90%

## 系统架构

- 感知层: OV5640相机、超声波传感器、光电开关等
- 决策层: 龙芯2K0301运行SLAM、RRT*路径规划、改进YOLOv8-Tiny识别
- 执行层: Cyclone IV FPGA完成图像预处理、逆运动学求解、电机/舵机PWM驱动

## 代码结构

- Verilog_HDL/       FPGA源码（图像预处理流水线、逆运动学求解器、PWM生成）
- algorithm/         龙芯端算法（YOLOv8训练/推理、ROS节点、SPI通信驱动）

## 快速开始

1. 硬件连接：龙芯2K0301与FPGA通过SPI对接，接好传感器、机械臂、底盘。
2. 龙芯端：安装Linux + ROS，编译SPI驱动，运行主控程序。
3. FPGA端：使用Quartus II下载工程到开发板。
4. 执行启动脚本，开始巡检采摘作业。
